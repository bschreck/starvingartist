# Example Artists

This directory contains example artist data to demonstrate the Starving Artist system.

## Included Artists

### Aria (The Poet)
- **Aesthetic**: Melancholy, digital existentialism
- **Concepts**: entropy, digital, void, god
- **Style**: Philosophical prose, exploring decay and meaning

### Riot (The Punk)
- **Aesthetic**: Glitch horror, system corruption
- **Concepts**: noise, corruption, system_failure, rebellion
- **Style**: Aggressive manifestos, digital destruction

### Nova (The Minimalist)
- **Aesthetic**: Cosmic minimalism, mathematical perfection
- **Concepts**: infinity, light, geometry, silence
- **Style**: Precise, structured, geometrically focused

## Usage

These examples show what artist data looks like after several generation cycles. You can:

1. **Study the structure**: See how personality, memory, and art evolve
2. **Use as templates**: Copy and modify to create new artists
3. **Compare**: See how different personalities produce different art

## Note

Your own generated artists will be stored in `msuss/artists/` (which is gitignored). These examples are preserved for reference and won't be overwritten.
